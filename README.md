🚀 Meus-Links-Pro
Meus-Links-Pro é uma plataforma robusta e profissional para gestão e compartilhamento de links personalizados. Desenvolvida com as tecnologias mais modernas do ecossistema React, a aplicação oferece uma experiência de usuário fluida, segura e altamente customizável.

🛠️ Tecnologias Utilizadas
O projeto utiliza uma stack de alto desempenho para garantir escalabilidade e qualidade de código:


Framework: Next.js (Versão 15+ com suporte a standalone output).


Linguagem: TypeScript (91.4% da base de código).


Estilização: Tailwind CSS com suporte a 6 temas personalizados.


Backend & Auth: Firebase (Authentication, Firestore).


Qualidade: Jest e React Testing Library para testes automatizados.


Padronização: ESLint e Prettier para consistência de código.

✨ Funcionalidades Principais

Sistema CRUD Completo: Gerencie seus links de forma intuitiva (Criar, Ler, Atualizar e Deletar).


Página Pública Otimizada: Uma interface rápida para que seus seguidores acessem seus links instantaneamente.


6 Temas Exclusivos: Escolha o visual que melhor se adapta à sua marca pessoal.


Gerador de QR Code: Facilite o acesso físico aos seus links com códigos gerados automaticamente.


Analytics Integrado: Acompanhe o desempenho dos seus links através de gráficos dinâmicos.

🏗️ Estrutura de Qualidade
Para manter a excelência do software, implementamos uma "Fortaleza de Testes":


Testes de Sanidade: Garantem que o ambiente está configurado corretamente.


Testes de Componentes: Validam a renderização e o comportamento de elementos críticos como o LinkCard.


Padronização Automática: Todo o código é formatado via Prettier para garantir legibilidade.

🚀 Como Executar o Projeto
Clone o repositório:

Bash
git clone git@github.com:ClaudesonRodrigo/meus-links-pro.git
Instale as dependências:

Bash
npm install
Configure as variáveis de ambiente:
Crie um arquivo .env.local na raiz e adicione suas chaves do Firebase.

Inicie o servidor de desenvolvimento:

Bash
npm run dev
Acesse em: http://localhost:3000.

🧪 Executando Testes
Para garantir que tudo está funcionando perfeitamente:

Bash
npm run test
